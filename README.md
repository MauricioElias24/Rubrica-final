# Iris Species Classification - Data Mining Final Project

## Purpose
Interactive dashboard for classifying Iris flower species using a Random Forest model. Built with Streamlit.

## Features
- **Model Metrics**: Accuracy, Precision, Recall, F1-Score
- **Real-time Prediction**: Input sepal/petal measurements to predict species
- **3D Visualization**: New sample positioned relative to the dataset
- **Histograms & Scatter Matrix**: Explore feature distributions

## How to Run

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the app:
   ```
   streamlit run Proyect.py
   ```

## Dataset
Iris dataset (150 samples, 4 features, 3 species):
- Iris setosa
- Iris versicolor
- Iris virginica

## Model
Random Forest Classifier (100 trees) trained on 70% of the data, tested on 30%.
